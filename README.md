# boxu-manageBo

博观

## 开发

只需要执行以下命令就可以在 `http://localhost:3333` 中看到页面：

```bash
pnpm run dev
```

## 构建

构建该应用只需要执行以下命令：

```bash
pnpm build
```

然后你会看到用于发布的 build 文件夹被生成。

## 预览

预览构建的应用只需要执行以下命令：

```bash
pnpm preview
```
